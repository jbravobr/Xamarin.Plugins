RegisterChallengeHandler
You can use this method to register a Challenge Handler in the client. You must use this method when you implement custom challenge handlers, or when you customize the Remote Disable / Notify Challenge Handler. Important: you must call this method at the beginning of your application after you initialize WLClient.
